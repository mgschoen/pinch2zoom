# Pinch 2 Zoom - Proof of Concept

### Demo

[http://dirty-kick.surge.sh/index.html](http://dirty-kick.surge.sh/index.html)

### Run locally

Install Parcel

```shell
$ yarn global add parcel-bundler
```

Install depencies
```shell
$ cd pinch2zoom-poc
$ yarn
```

Start dev server

```shell
$ yarn start
```

### Build and deploy

Build everything once:

```shell
$ yarn run build
```

Deploy to surge.sh:

```shell
$ yarn run deploy
```
